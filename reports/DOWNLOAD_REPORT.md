## Download Method

1. **Initial download**: PowerShell `Invoke-WebRequest` for all 33 files listed in
   `offline.appcache` and the main `index.html`.
2. **Verification**: Playwright (Chromium headless) loaded the game, simulated
   gameplay (mouse clicks, keyboard input, shooting), and recorded all 21 actual
   network requests. All returned HTTP 200.
3. **Result**: All resources identified in the appcache manifest are the exact
   same set as those loaded dynamically. No additional hidden resources.
4. **Static files**: 8 files referenced in HTML (icons, manifests, service worker)
   were not dynamically requested but were also downloaded.

## Modifications for Local Operation

1. **`app/index.html`**: Removed `manifest="offline.appcache"` attribute from
   `<html>` tag to avoid deprecated AppCache warnings. All paths were already
   relative — no further path changes needed.
2. **No other files were modified.** The game uses relative paths throughout.
